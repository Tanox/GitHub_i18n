### 1.2 核心翻译流程详解

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                         elementTranslator.translateElement()                         │
└─────────────────────────────────────────────────────────────────────────────────────┘

                                    ┌─────────────────┐
                                    │ 接收 DOM 元素   │
                                    └────────┬────────┘
                                             │
                                             ▼
                              ┌──────────────────────────────┐
                              │ 遍历子节点 (childNodes)      │
                              └──────────────┬───────────────┘
                                             │
                    ┌────────────────────────┼────────────────────────┐
                    │                        │                        │
                    ▼                        ▼                        ▼
          ┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
          │  TEXT_NODE      │      │  ELEMENT_NODE   │      │  其他节点       │
          │  (文本节点)      │      │  (元素节点)      │      │  (跳过)         │
          └────────┬────────┘      └────────┬────────┘      └─────────────────┘
                   │                        │
                   ▼                        ▼
        ┌─────────────────────┐    ┌─────────────────────┐
        │ 1. 获取文本内容      │    │ 1. 移除子节点        │
        │    node.nodeValue   │    │ 2. 临时保存到        │
        │                     │    │    DocumentFragment  │
        │ 2. 预检查翻译匹配     │    │                     │
        │ ┌─────────────────┐│    │ 3. 递归调用          │
        │ │ getTranslatedText││    │    translateElement  │
        │ │ 检查是否有对应翻译 ││    │    处理子元素        │
        │ └────────┬────────┘│    │                     │
        │          │         │    │ 4. 重新添加子节点     │
        │          ▼         │    │    回原父元素         │
        │ ┌─────────────────┐│    └─────────────────────┘
        │ │ 无匹配?         ││
        │ │ 直接返回false   ││
        │ │ 不修改DOM       ││
        │ └─────────────────┘│
        │          │         │
        │          ▼         │
        │ ┌─────────────────┐│
        │ │ 有匹配!         ││
        │ │ 记录 hasTranslat-││
        │ │ ableContent=true││
        │ └────────┬────────┘│
        │          │         │
        │          ▼         │
        │ ┌─────────────────┐│
        │ │ 创建文本片段    ││
        │ │ 用翻译文本替换   ││
        │ │ 原文本内容      ││
        │ └────────┬────────┘│
        │          │         │
        └──────────┼─────────┘
                   │
                   ▼
        ┌─────────────────────┐
        │ 翻译完成，返回结果   │
        │ hasTranslation      │
        └─────────────────────┘
```

### 1.3 词典匹配流程 (Trie 树优化)

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                         dictionaryManager.getTranslatedText()                       │
└─────────────────────────────────────────────────────────────────────────────────────┘

       ┌──────────────────────────────────────────────────────────────────────────┐
       │                           词典匹配查询流程                                  │
       └──────────────────────────────────────────────────────────────────────────┘

    用户文本输入
         │
         ▼
┌─────────────────┐
│  1. 缓存检查     │ ◄── LRU 缓存 (cacheManager)
│     cache.get()  │     快速返回已翻译结果
└────────┬────────┘
         │ 缓存未命中
         ▼
┌─────────────────┐
│  2. 精确匹配     │
│   dictionary.get│
│   (完整文本)     │
└────────┬────────┘
         │ 找到
         ▼
    返回翻译结果
         │
         ▼
┌─────────────────┐
│  缓存写入       │
│  cache.set()    │
└─────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────┐
│                         partialTranslator.matchPartial()                            │
└─────────────────────────────────────────────────────────────────────────────────────┘

    用户文本: "This repository is amazing"
         │
         ▼
┌─────────────────┐
│  Trie 树查询    │
│                 │
│    ┌───┐        │
│    │ T │h|i|s   │
│    └───┘        │
│      └─r|e|p|o  │
│        └─s|i|t  │
│          └─ory │
│                 │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  匹配结果:       │
│  "This" → "这"  │
│  "repository" → │
│    "仓库"       │
│  "is" → "是"    │
│  "amazing" →    │
│    "令人惊叹的"  │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  组装翻译文本    │
│  "这 仓库 是    │
│   令人惊叹的"   │
└─────────────────┘
```
