Design System / Prototype

让 GitHub 以
中文的节奏呼吸。

这是一套为 GitHub Chinese 简体中文插件量身定制的设计系统。它承袭 GitHub 的深色美学、精确排版与克制交互，同时以中文衬线字体、语义色彩 与细腻动效，塑造属于插件自身的独立识别。

设计原则

Design Principles · 四条核心准则

  • 克制胜于花哨——色彩与装饰服务于信息层次,而非喧宾夺主。
  • 一致胜于独创——遵循 GitHub 原生交互节奏,让插件如原生般自然。
  • 中文优先——以中文阅读节奏定义字号、行距与对齐方式。
  • 可演化——所有令牌与组件均可替换,系统随项目一同生长。
